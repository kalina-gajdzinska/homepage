Tydzień IV

Terminal - wiersz poleceń na Windows ->CMD
GIT - system kontroli wersji.
Commituj zmiany! git status / git add. / git commit-m"opis zmiany" / git push
Konto na GitHubie to programistyczna wizytówka.
<div> - służy do opakowywania, nie ma znaczenia semantycznego.
dispay:block - ustawia elementy jeden pod drugim.
display:inline - ustawia elementy obok siebie.
display:inline-block - nadaje dodatkowe własności, takie jak: width i height.
max-width i max-height - ustawia maksymalną szerokość lub wysokość danego elementu.
Jednostki: px - piksele, % - procenty, vw - viewport width, vh - viewport heigh
padding - wewnętrzny margines
border - obramowanie (border-radius - zaokrąglenie)
margin - margines (odsunięcie od innych elementów)
top, right, bottom, left
auto - wyśrodkowuje, dzieli na pół (element musi mieć zdefiniowaną szerokość)
box sizing: content-box: stałe wymiary elementu, nie wpływa na padding, border i margin
border-box: dostosowuje wymiary elementu, razem z paddingiem, border i margin.
Position:static/relative/absolute
z-index - element z wyższym z-indexem jest widoczny.
position:fixed - przykleja element
position:stick - przykleja do elementu, ma wpływ na treść
float - służy do opływania elementów
